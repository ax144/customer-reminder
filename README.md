# 📋 客户关怀提醒系统

> 基于扣子+Supabase + GitHub Actions + 飞书的智能客户关怀自动推送系统

---

## ✨ 功能特性

| 🌅 上午推送 | 🌙 下午推送 |
|------------|------------|
| 满7天及超过7天未联系的客户提醒 | 今日已联系客户总结 |
| 帮助跟进重要客户 | 记录每日工作成果 |

- ✅ **防重复机制** - 多次触发只推送一次
- ⏰ **多时间点覆盖** - 应对 GitHub Actions 延迟
- 📊 **Supabase 数据存储** - 安全可靠
- 🔔 **飞书消息推送** - 实时提醒

---

## ⏰ 推送时间

| 📅 时段 | ⚙️ 设置时间 | 🎯 预计推送 |
|--------|------------|------------|
| 🌅 上午 | 8:28, 8:56, 9:24 | ~8:56 - 9:52 |
| 🌙 下午 | 16:28, 16:56, 17:24 | ~16:56 - 17:52 |

> 💡 GitHub Actions 延迟约 20-30 分钟，设置时间会延后执行
